# Superdex 200 pg 层析柱（SK 16/70）装柱操作流程

## 1. 填料准备

1. **计算填料用量**  
   根据最终柱体积和填料压缩因子计算所需悬浮液体积。  
   - 填料压缩因子：1.15  
   - 柱体积：120 mL  
   - 悬浮液所需体积：120 × 1.15 = 138 mL（示例中为 184 mL）  

2. **静置沉淀**  
   建议采用静置沉淀法，确定胶悬液浓度。  
   - 大部分机翼填料以 75% 浓度储存在 20% 乙醇中。

3. **溶液置换**  
   - 通过真空抽滤，将 20% 乙醇更换为装柱所需溶液（如水）。  
   - 重复三次。

4. **最终悬浮**  
   - 用装柱缓冲液重新悬浮填料，并用量筒定容到所需体积（如 184 mL）。  

---

## 2. 装柱前准备

1. **工具准备**  
   - 填料、空柱、装柱器  
   - 底端配件、顶部适配器、注射器连接器、滤膜工具  
   - 蒸馏水、20% 乙醇  
   - 堵头  

2. **设备准备**  
   - 层析系统（如 Akta Pure）  
   - 检查系统压力、流速、UV 和电导率监测器、Unicorn 软件  

3. **柱体检查**  
   - 检查玻璃柱端部，确保无缺口  
   - 柱子和装柱器用 20% 乙醇润洗  
   - 检查筛网孔径与填料粒径匹配  
   - 确保底端部件和顶部适配器连接牢固  

---

## 3. 装柱操作流程

1. **柱底准备**  
   - 用注射器推入 20% 乙醇排空气，让乙醇停留在滤网上  
   - 连接底端柱头到柱子  
   - 固定柱子（助教或铁架台）  

2. **液体加入**  
   - 向柱中加入 20% 乙醇或蒸馏水至几厘米高度  
   - 连接装柱器，确保柱管垂直  

3. **装柱器设置**  
   - Unicorn 软件设定压力报警（如 Superdex 200 pg，0.5 MPa）  
   - 使用玻璃棒将胶悬液倒入柱管，贴壁倒入  
   - 用装柱缓冲液补足装柱器上端空余体积  
   - 盖上装柱器顶盖  

4. **排气泡**  
   - 使用 Pump Watch 功能排出设备及管路中的气泡  
   - 底部放废液缸，底端拧开，将管路与装柱器顶部液滴对液滴连接  

5. **装柱运行**  
   - 按填料说明书推荐流速运行  
   - 第二阶段可使用更高压力压实填料  
   - 观察胶面下降至装柱器以下并稳定  
   - 用记号笔标记胶面  

6. **装柱器拆卸**  
   - 停止流速  
   - 底部堵头封闭  
   - 拆卸装柱器顶部  
   - 用滴管补足柱头液面  
   - 顶端适配器推入 20% 乙醇排空气，停留滤网上  
   - 适配器缓慢下降至胶面，液体从柱头排出  
   - 上端螺纹紧固  

7. **最终连接与调节**  
   - 重新将管路与柱头液滴对液滴连接  
   - 打开底部堵头，继续运行流速  
   - 如胶面变化可调节适配器高度  

---

## 4. 柱效测定

1. **样品准备**  
   - 方法 1：1% 丙酮，水为流动相，检测 280 nm UV  
   - 方法 2：0.8 mol/L NaCl，0.4 mol/L NaCl 为流动相，检测电导率  
   - 上样量：1% 柱体积（120 mL × 1% = 1.2 mL）  

2. **测定操作**  
   - 低流速 30 cm/h，先上后下方式连接管路  
   - 将样品通过注射器推入 loop 环  
   - Unicorn 软件 Master Editor 设置参数：压力报警、缓冲液入口、流速、平衡体积、上样体积  

3. **结果分析**  
   - 积分电脑曲线，输入柱高值  
   - 查看 HETP 和对称性  
   - 合格标准：对称性 0.8–1.5，理论塔板数符合说明书要求  

---

## 5. 暂时不使用或拆卸柱子

1. **封闭柱头**  
   - 两端用堵头封闭（先下后上方式）  

2. **拆卸操作**  
   - 拧松上端黑色螺纹口，移除顶部适配器  
   - 将柱子链接到 Akta 系统  
   - 软件设置对应压力和流速，将填料顶出  
   - 填料和空柱可重新使用
